- [x] Game timer

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

- Node.js
- Git

### Installation

```bash
git clone https://github.com/Sanjero20/geo-trainer.git
cd geo-trainer
npm install
```

### Usage

```bash
npm build && npm start
```
